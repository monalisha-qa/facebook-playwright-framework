# Myntra Playwright Automation Project

This is a UI automation framework using Playwright with Java and TestNG. It automates Myntra.com features like product search, page navigation, and more.

## Tech Stack
- Java
- Playwright
- TestNG
- Maven
- POM (Page Object Model)
- GitHub

## How to Run
```bash
mvn exec:java -e -Dexec.mainClass=com.microsoft.playwright.CLI -Dexec.args="install"
mvn test
